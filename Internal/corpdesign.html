Fonts

 

The font Helvetica Neu is used and used as a publication font at MEC Electronics. The Helvetica Neu font has several styles.

This is a font purchased for a fee.

 

In order not to complicate the use of the font in everyday use, it was agreed to use the Calibri font for open files and email correspondence.
